# BoneSnippets

针对阿里智能 [Bone](https://open.aliplus.com/bone/docs/bone_flow.html) 框架的 Snippet 插件（暂时只支持Visual Studio Code）。

## 快捷命令
* BoneRouter
* BoneController
* BoneView
* BoneBaseService
* BoneDeviceService

## 支持 React Native 生命周期函数
* getInitialState
* componentWillMount
* componentDidMount
* componentWillReceiveProps
* shouldComponentUpdate
* componentWillUpdate
* componentDidUpdate
* componentWillUnmount
